⚡ Установка за 60 секунд

Простые шаги — и твой форум для испанского готов

1

GitHub аккаунт

Зарегистрируйся бесплатно на github.com

2

Открой шаблон

Перейди на feekool.github.io — это наш готовый шаблон для испанского

3

Создай свой репозиторий

Нажми "Use this template" → "Create a new repository"
Назови, например: mi-curso-espanol или spanish-club

4

Включи GitHub Pages

Settings → Pages → Branch: main → Save
Через 1-2 минуты форум будет доступен по ссылке вида:
https://твой-ник.github.io/ми-курсо-эспаньол

5

🇪🇸 Настрой под испанский!

Отредактируй файлы в своём репозитории:
• index.md — приветствие «¡Bienvenidos!»
• config.json — названия разделов на испанском
• Добавь уровни: A1, A2, B1...
Готово! ¡Vamos!

💡 Pro tip: В шаблоне уже есть пример структуры: «Gramática», «Vocabulario», «Práctica», «Cultura». Просто раскомментируй и наполняй!
